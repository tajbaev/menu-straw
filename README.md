# 🍫 Choco Garden - Сайт Меню

Красивый и интерактивный сайт для просмотра меню кафе "Choco Garden".

## 📋 Функциональность

✅ **Фильтрация по категориям** - Просмотрите товары по категориям (Кофе, Шоколадные напитки, Десерты, Смузи)
✅ **Карточки товаров** - Красивые карточки с названием, описанием и ценой
✅ **Модальное окно** - Подробная информация о блюде при клике на "Подробнее"
✅ **Адаптивный дизайн** - Отлично смотрится на всех устройствах
✅ **Интерактивность** - Плавные анимации и переходы

## 📁 Структура файлов

```
choco garden/
├── index.html        # Главная HTML страница
├── style.css         # Стили и оформление
├── script.js         # Интерактивность и логика
├── menu-data.js      # Данные меню
└── README.md         # Этот файл
```

## 🚀 Как использовать

1. **Откройте файл `index.html`** в браузере
2. **Фильтруйте товары** - Нажимайте на кнопки категорий
3. **Просмотрите детали** - Кликайте "Подробнее" для полной информации

## 🎨 Оформление

- **Цветовая схема**: Шоколадные и золотые тона
- **Шрифт**: Segoe UI для лучшей читаемости
- **Иконки**: Используются эмодзи для визуальности

## 📝 Как добавить товар

Отредактируйте файл `menu-data.js` и добавьте объект в массив `menuData`:

```javascript
{
    id: 10,
    title: 'Название товара',
    category: 'coffee', // или 'chocolate', 'desserts', 'smoothies'
    price: '200 ₽',
    description: 'Описание товара',
    image: 'URL или data URI'
}
```

## 🎯 Категории

- `coffee` - Различные виды кофе
- `chocolate` - Шоколадные напитки
- `desserts` - Десерты и выпечка
- `smoothies` - Смузи и коктейли
